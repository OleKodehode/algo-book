# README for Psuedocode folder

This folder contains all the psuedocode I thought were relevant to note down while I read a book on programming algorithms foundations. It's not a 1-to-1 from the book, went with more of a JS style psuedocode.

I sort of used an arbitrary format for the psuedocode files, using .psu and setting up VSCODE to utilize the generic pseudocode formatting supplied by the extension [Pseudocode](https://marketplace.visualstudio.com/items?itemName=willumz.generic-pseudocode) by Willumz.
